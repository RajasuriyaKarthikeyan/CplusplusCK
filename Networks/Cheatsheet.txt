### üì∂ **1. Network Models**

#### OSI Model (7 Layers)

| Layer           | Function                    | Example Protocols |
| --------------- | --------------------------- | ----------------- |
| 7. Application  | User interface              | HTTP, FTP, SMTP   |
| 6. Presentation | Data translation/encryption | SSL, JPEG         |
| 5. Session      | Manage sessions             | NetBIOS, RPC      |
| 4. Transport    | Reliable delivery           | TCP, UDP          |
| 3. Network      | Routing                     | IP, ICMP          |
| 2. Data Link    | MAC addressing              | Ethernet, PPP     |
| 1. Physical     | Media, signals              | Cables, Hubs      |

#### TCP/IP Model (4 Layers)

| Layer          | OSI Equivalent | Protocols      |
| -------------- | -------------- | -------------- |
| Application    | 7, 6, 5        | HTTP, FTP, DNS |
| Transport      | 4              | TCP, UDP       |
| Internet       | 3              | IP, ICMP       |
| Network Access | 2, 1           | Ethernet, ARP  |

### üì° **3. IP Addressing**

#### IPv4 Format

* **32-bit address**: e.g. `192.168.1.1`
* **Classes**:

  * A: 1.0.0.0 ‚Äì 126.255.255.255 (Default Subnet: /8)
  * B: 128.0.0.0 ‚Äì 191.255.255.255 (Default Subnet: /16)
  * C: 192.0.0.0 ‚Äì 223.255.255.255 (Default Subnet: /24)

#### Private IP Ranges

| Class | Range                         |
| ----- | ----------------------------- |
| A     | 10.0.0.0 ‚Äì 10.255.255.255     |
| B     | 172.16.0.0 ‚Äì 172.31.255.255   |
